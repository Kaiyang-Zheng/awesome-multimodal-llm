**有很多同学反应不太了解什么是3-sigma，简单来说就是，**

**假设原始数据的某个分析维度服从正态分布且存在随机误差，我们可以基于Analyzer产出的stats，在该维度上**

**把算子过滤的上下界设为三倍标准差。**

$$P(|x-\mu|>3 \sigma) \leq 0.003$$

**这里可以参考data- juicer的解释和用法**：https://github.com/modelscope/data-juicer/blob/main/tools/hpo/README\_ZH.md

# **9.6.2 多样性探索**

> 传统&#x7684;**`SFT+DPO`**&#x5BF9;齐这块，`Prompt`多样性是非常重要的，这也是为什么出上一篇笔记的原因，这次实验我试了两种不同的多样性探索实验。

## 实验设置一

**核心**：**采用KMQ算法，利用k-means和k-center思想，选用`KMeansRandom`进行数据选择。**

* **对`KMeansRandom`进行初始化，设置聚类数K，并创建AutoEncoder对象。**

  ```python
  def __init__(self, dataset, data_config, method_config, encoder_config=None, K=1024):
          super().__init__(dataset, data_config, method_config, encoder_config)
          self.K = K
          self._is_raking = False
          self.encoder = AutoEncoder(encoder_config)
  ```

* **实现K-means聚类并选择核心集索引**

  * 获取数据集的嵌入向量

    ```python
    embeddings = self.encoder.get_embeddings(self.dataset, self.data_config)
    print("Performing K-means clustering...")
    d = embeddings.shape[1]
    ```

  * 执行K-means聚类，将嵌入向量分配到最近的聚类中心

    ```python
    kmeans = faiss.Kmeans(d, self.K, niter=300, verbose=True, nredo=5, gpu=True)
    kmeans.train(embeddings)
    distances, indices = kmeans.index.search(embeddings, 1)
    indices = indices.reshape(-1)
    final_indices = np.array([], dtype=np.int64)
    ```

  * 对每个聚类，随机选择一定数量的样本索引，确保每个聚类都有代表

    ```python
    for i in range(self.K):
        indices_i = np.where(indices == i)[0]
        size = np.minimum(int(self.coreset_size/self.K), len(indices_i))
        indices_i = np.random.choice(indices_i, size=size, replace=False)
        final_indices = np.concatenate((final_indices, indices_i))
    ```

* **对句子的质量和复杂度进行打分排序**

  * 定义打分器

    ```python
    def infer_score(self, user_input: str):
            max_length = 512
            if self.is_vllm:
                outputs = self.llm.generate(user_input, self.sampling_params, use_tqdm = False)
                score_template = np.array([1,2,3,4,5,6])

            else:
                input_ids = self.tokenizer.encode(user_input, return_tensors = "pt")
                outputs = self.model.generate(input_ids, max_length = max_length, num_return_sequences = 1, return_dict_in_generate = True, output_scores = True)
                logprobs_list = outputs.scores[0][0]            
            score_logits = []
            score_template = np.array([1,2,3,4,5,6])
            for k in self.id2score:
                score_logits.append(logprobs_list[k])
            score_logits = np.array(score_logits)
            score_npy = softmax(score_logits, axis=0)
            score_npy = score_npy * score_template
            score_npy = np.sum(score_npy, axis=0)
    ```

## 实验设置二

**核心**：**KNN聚类，类内平均样本相似度作为多样性权重（反过来），结合质量权重，最后根据权重采样。**

* **计算`KNN`权重**

  * 加载指定语言的数据集嵌入向量，创建向量数据库，并计算每个数据集的多样性权重。

  ```python
  def KNN_load_weight_meta(meta_data_file,language):
      meta_data = [json.loads(i) for i in open(meta_data_file)]
      embedding_lists = []
      for dataset in meta_data:
          if dataset['language'] != language:
              continue
          embedding_file = dataset['file_name'].rsplit('.', 1)[0] + '.npy'
          #加载一个数据集的embeddings
          embeddings = np.load(embedding_file).astype('float32')
          embedding_lists.append(embeddings)
      #创建向量数据库
      faissWithNoModelEncode = FaissWithNoModelEncode(embedding_lists)
      richness = {}
      for dataset in meta_data:
          if dataset['language'] != language:
              continue
          embedding_file = dataset['file_name'].rsplit('.', 1)[0] + '.npy'
          #加载一个数据集的embeddings
          embeddings = np.load(embedding_file).astype('float32')
          #搜索向量数据库
          print(f'searching {dataset["file_name"]}...')
          D,I = faissWithNoModelEncode.search_index(embeddings,256)
          #生成多样性权重
          dis = np.mean(1-D,axis=1)
          dis = np.nan_to_num(dis)

          richness[dataset['file_name']] = dis_minmax
          richness_file = dataset['file_name'].rsplit('.', 1)[0].replace('v4_1','v4_1_reverse_test') + '_diversity_confirm.json'
          json.dump({dataset['file_name']:dis_minmax},open(richness_file,'w'))
      return richness
  ```

  ```python
  def meta_rank(folder,experiment_name,sample_nums):
      
      folder_write = ''
      if not os.path.exists(folder_write):
          os.makedirs(folder_write)
      if not folder.endswith('/'):
          folder += '/'
      meta_data = [json.loads(i) for i in open(folder + 'meta_data.jsonl')]
      sample_index = dict()
      for dataset in meta_data:
          language = dataset['language']
          dataset_name = dataset['file_name'].split('/')[-1]
          sample_rate = float(dataset['sample_rate'])
          key = dataset['file_name']
          logger.info("sampling file_name={}".format(key))
          print(f'sample_rate:{sample_rate}')
          ###语义相似度+质量4个指标抽样###
          if language == 'zh':
              beta = 0.5
              with open(folder + 'quality_sort_{}'.format(dataset_name)) as fr:
                  quality = json.load(fr)
              with open(dataset['file_name'].rsplit('.', 1)[0] + '_diversity.json') as fr:
                  diversity = json.load(fr)

              weights_semantic = np.array(diversity[key])

              weights_quality = np.array(quality[key])
              weights_quality = np.max(weights_quality) - weights_quality
              weights_quality = MinMaxScaler().fit_transform(weights_quality.reshape(-1, 1)).reshape(-1, )
              weights = beta * weights_semantic + (1 - beta) * weights_quality

              sample_num = int(sample_nums * sample_rate)
              intervals = max(2, sample_num // 10)

                  sample_type = 'linear'

                  sample_type = 'equal'
              if sample_num / len(weights) < 1:
                  drop_rate = min((1 - (sample_num / len(weights))) * 0.3, 0.2)
              drop_rate = 0.0
              indexs = sampleWithWeights.sample_with_weights(weights=weights, sample_num=sample_num,
                                                             sample_type=sample_type, intervals=intervals,
                                                             drop_rate=drop_rate,
                                                             complete_sample=True)
              print(np.mean(weights[weights >= 0.0]), np.mean(weights[indexs]))
              sample_index[key] = list(map(int, list(indexs)))
          # 随机抽取英文数据
          else:
              random.seed(10)
              sample_num = int(sample_nums * sample_rate)
              num = int(dataset['data_nums'])

                  index = [i for i in range(num)]
  ```
